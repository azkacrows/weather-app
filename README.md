# weather-app (latihan mandiri)

## Deskripsi Proyek

Proyek ini merupakan latihan mandiri dalam membangun sebuah aplikasi cuaca sederhana yang menampilkan informasi cuaca terkini. Aplikasi ini menggunakan OpenWeatherMap API untuk mengambil data cuaca dan menampilkan informasi tersebut di antarmuka pengguna. Tujuan dari proyek ini adalah untuk mempraktikkan keterampilan dalam menggunakan API dan mengintegrasikan data eksternal ke dalam aplikasi web.

## Struktur Proyek

-   `index.html`
-   `style.css`
-   `app.js`
-   `icons/`

### Penjelasan Folder dan File

-   `index.html`: Berisi struktur HTML dari aplikasi cuaca, termasuk elemen-elemen antarmuka pengguna yang diperlukan.
-   `style.css`: Berisi gaya dan desain visual untuk aplikasi cuaca, menentukan tata letak dan penampilan elemen di halaman.
-   `app.js`: Berisi logika JavaScript untuk berinteraksi dengan OpenWeatherMap API, mengambil data cuaca, dan memperbarui antarmuka pengguna.
-   `icons/`: Folder yang berisi ikon-ikon yang digunakan dalam aplikasi untuk menampilkan informasi cuaca dengan cara yang lebih menarik.

## Fitur Utama

1. **Informasi Cuaca Terkini**: Menampilkan suhu, kondisi cuaca, dan informasi relevan lainnya menggunakan data dari OpenWeatherMap API.
2. **Desain Responsif**: Desain aplikasi yang responsif dan mudah digunakan di berbagai perangkat.
3. **Penggunaan API**: Mengintegrasikan OpenWeatherMap API untuk mendapatkan data cuaca secara real-time.

## Cara Menggunakan

1. **Clone Repository**:

    ```sh
    git clone https://github.com/username/weather-app.git
    ```

2. **Masukkan API Key**:

    Edit file `app.js` dan masukkan API key OpenWeatherMap yang valid pada bagian yang telah ditentukan.

3. **Jalankan Aplikasi**:

    Buka file `index.html` di browser untuk menjalankan aplikasi cuaca.
